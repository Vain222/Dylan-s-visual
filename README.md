## HTML File Preview Guide

Since GitHub does not directly render HTML files in the repository by default (only the source code is displayed), you can quickly preview the actual effect of HTML files through the 'htmlpreview.github.io' tool.

Splice the copied original URL after 'https://htmlpreview.github.io/?` to form a complete preview link, format example: https://htmlpreview.github.io/?https://github.com/Vain222/Dylan-s-visual/blob/main/MATH500-visual-Qwen3-1.7B.html
